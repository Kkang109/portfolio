PROJECT . 01

포트폴리오 웹사이트

2026. 05 - 2026. 06 (개인 프로젝트)

LAUNCH WEBSITE

OVERVIEW

음악 플레이리스트라는 독창적인 컨셉을 차용하여, 지루할 수 있는 포트폴리오 탐색 과정에 인터랙티브한 재미를 부여한 디자이너/퍼블리셔 맞춤형 웹사이트입니다.

UI/UX DESIGN PROCESS

파스텔 블루톤을 베이스로 설정하여 청량하고 트렌디한 느낌을 주었으며, 모든 모서리에 20px 이상의 라운딩을 적용하여 몽글몽글하고 친근한 그래픽 스타일을 구축했습니다.

UI Design System & Wireframe Zone

PUBLISHING POINT

100vh 풀스크린 레이아웃을 사용하여 스크롤 파편화를 방지했고, Flexbox와 CSS Grid를 혼합하여 화면 해상도 변화에도 깨지지 않는 견고한 웹 앱 UI를 마크업했습니다.
